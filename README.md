# Naufal Ilmi Portfolio

Personal portfolio for Naufal Ilmi, focused on cloud, network security, SOC monitoring, infrastructure, and AI automation.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- lucide-react
- Vercel-ready deployment

## Local Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

## Production Build

```bash
npm run lint
npm run build
```
